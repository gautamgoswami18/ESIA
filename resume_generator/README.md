# Resume Generator

This module generates professional ATS-friendly resumes for all employees.

## Input Files

```
data/

employee.xlsx

employee_skills.xlsx

employee_projects.xlsx

employee_certifications.xlsx

employee_training.xlsx
```

## Output

```
documents/resumes/

100 Resume PDFs
```

```
output/

resume_metadata.xlsx

resumes.zip
```

## Run

```bash
python generate_resumes.py
```
